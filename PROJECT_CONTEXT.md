Analyze websites and improve marketing/conversion systems.

Audience:

- founders

- marketers

- businesses with traffic and sales teams

Main UX goals:

- trust in first seconds

- premium SaaS feeling

- low friction

- clean interface

Tech stack:

- React

- Use async/await

- Avoid duplicated logic

UI Style:

- modern SaaS

- minimal

- smooth transitions

- fast loading

- clean spacing

- no clutter

Important:

Always explain risky changes before applying them.

Modify only necessary parts of files.
